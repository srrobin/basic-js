box-sizing: content-box vs border-box 

        Calculation of dimension:
        -------------------------
        let, width =300px , height=100px ,padding= 5px, margin=5px border=5px
        --> for content-box:::
        width = border(both side) + padding(both side) +border(both side)+ width,
        300px = (5 + 5)+(5 + 5)+(5 + 5)+ 300  = 330px 

        height =  border(both side) + padding(both side) +border(both side)+ height
        300px = (5 + 5)+(5 + 5)+(5 + 5)+ 100  = 130px 

        --> for border-box::: 
        width = border(both side) + padding(both side) +border(both side)+ width,
        300px = (5 + 5)+(5 + 5)+(5 + 5)+ 270  = 300px 
                                      -------   
        height =  border(both side) + padding(both side) +border(both side)+ height
        100px = (5 + 5)+(5 + 5)+(5 + 5)+ 70  = 100px  
                                      ------
